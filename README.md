# phone-compare
Phone Compare is a personal project made to make it easier to compare the total cost of taking a phone out on contract for 24 months with a network provider against buying a phone outright (with or without a seperate sim only contract at a fixed price). [Click Here](https://zainy2401.github.io/phone-compare/www) to go to website!

## Help and other
For a short guide on how to use the website; on the **left** (**top** on mobile phones) side, enter a value for the up front cost, if 0 then leave at 0, then enter the monthly price, then enter the price incerases for the following 2 Aprils, ~if there are no price increases then leave the last 2 input fields the same as the "monthly price" field otherwise it will mess up the calculation by assuming the contract price changes to 0 per month!~ on the **right** (**bottom** on mobile phones) side, enter a value for the cash price field and optionally add the monthly price value for the last field to calculate a seperate sim only for the duration of 24 months alongside the cash price for total cost.
